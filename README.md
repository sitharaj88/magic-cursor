Open Settings (`Cmd+,` / `Ctrl+,`) and search for "Magic Cursor":

| Setting | Description | Default |
|---------|-------------|---------|
| `magicCursor.enableTrails` | Enable/disable cursor trails | `true` |
| `magicCursor.enableParticles` | Enable/disable typing sparkles | `true` |
| `magicCursor.cursorStyle` | Choose trail style: `rainbow`, `neon`, `aurora`, `sunset` | `rainbow` |
